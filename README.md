# Apache OFBiz (apache-ofbiz)
Apache OFBiz is an open-source enterprise resource planning system with a suite of business applications for CRM, e-commerce, supply chain management, manufacturing, and more. It provides a service-oriented architecture with web service APIs.

## APIs

### Apache OFBiz
OFBiz provides REST and XML-RPC service APIs for ERP/CRM functions, an entity engine API for data access, a service engine API for business logic, and a widget framework API for UI customization.
